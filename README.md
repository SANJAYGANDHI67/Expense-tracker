# 💰 Expense Tracker

A modern Expense Tracker web application built using HTML, CSS, JavaScript, Supabase, and Chart.js.

The application helps users manage expenses, monitor spending habits, track budgets, generate reports, and visualize expense data.

## 🚀 Live Demo

https://sanjaygandhi67.github.io/Expense-tracker/

---

## 📌 Features

### 1. Expense Management

- Add new expenses
- Delete expenses
- Store expenses in Supabase cloud database
- Real-time expense list updates
- Automatic total expense calculation

### 2. Budget Management

- Set monthly budget
- Save budget locally
- Calculate remaining budget automatically
- Track budget usage percentage
- Budget exceeded warning

### 3. Search & Filter

- Search expenses by name
- Filter expenses by category
- View category-specific expenses instantly

### 4. Sorting System

Sort expenses by:

- Latest First
- Oldest First
- Highest Amount
- Lowest Amount

### 5. Date Tracking

- Save expense date
- Display expense date in expense cards
- Use dates for reporting and analysis

### 6. Expense Reports

Generate reports for:

- Current Month
- Any Selected Month
- Last 7 Days

Report includes:

- Category-wise spending
- Total spending amount

### 7. Data Visualization

Interactive Pie Chart showing:

- Food Expenses
- Travel Expenses
- Shopping Expenses
- Bills Expenses
- Other Expenses

### 8. CSV Export

Download complete expense history as:

```csv
Name,Amount,Category,Date
Gym,1200,Other,2026-06-01
Snacks,200,Food,2026-06-16
```

Useful for:

- Excel
- Google Sheets
- Personal finance records

### 9. Responsive Design

- Desktop friendly
- Tablet friendly
- Mobile friendly

### 10. Cloud Database Integration

Powered by Supabase:

- Cloud storage
- Persistent data
- CRUD operations
- Online accessibility
